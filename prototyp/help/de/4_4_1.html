Aufgaben: Ansicht: Neue Aufgaben

Diese Ansicht zeigt Ihnen alle wichtigen Details zu einer neuen Aufgabe an.

Klicken Sie auf den Button "Rückfrage", um eine Rückfrage an den Aufgabensteller zu stellen. Es erscheint das Dialogfenster "Rückfrage stellen".

Klicken Sie auf den Button "Nachricht", um eine Nachricht an den Bearbeiter einer aktiven Aufgabe zu senden. Es erscheint das Dialogfenster "Nachricht".

Klicken Sie auf den Button "Weiterleiten", um eine  Aufgabe an einen anderen Mitarbeiter zu delegieren. Es erscheint das Dialogfenster "Weiterleiten".

Klicken Sie auf den Button "Fälligkeit", um den Fälligkeitstermin einer neuen oder aktiven Aufgabe zu verändern. Es erscheint das Dialogfenster "Fälligkeit ändern".

Klicken Sie auf den Button "Priorität", um die Priorität einer neuen oder aktiven Aufgabe zu ändern. Es erscheint das Dialogfenster "Priorität ändern".

Klicken Sie auf den Button "Neuer Kommentar", um einen Kommentar zu einer einzufügen. Es erscheint das Dialogfenster "Kommentar erstellen".

Klicken Sie auf den Button "Übernehmen", um eine neue oder schon aktive Aufgabe von einem anderen Benutzer übernommen werden.

Klicken Sie auf den Button "Abhaken", um eine aktive Aufgabe abzuhaken. Es erscheint das Dialogfenster "Aufgabe abhaken".

Klicken Sie auf den Button "Reaktivieren", um eine bereits erledigte Aufgabe zu reaktivieren. Es erscheint das Dialogfenster "Aufgabe reaktivieren", welches mit dem Dialogfenster "Neue Aufgabe erstellen" identisch ist.
