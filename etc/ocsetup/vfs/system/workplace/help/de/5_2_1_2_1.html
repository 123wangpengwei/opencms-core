Administration: Benutzerverwaltung: Gruppe: Neu

Tragen Sie in die Eingabefelder im Dialogfenster "Gruppe neu anlegen" ein:

Gruppenname:Eingabe des Gruppennamens. Wichtig! Es ist nicht möglich, den Gruppenname nachträglich zu ändern.
Beschreibung: Eingabe einer näheren Beschreibung der Gruppe, z.B. Aufgabenbereich.
Übergeordnete Gruppe: Wählen Sie aus, ob die Gruppe die Benutzerrechte einer bereits existierenden Gruppe erhalten soll.

Diese Gruppe definieren als:
Projektleiter: Aktivieren Sie diese Checkbox, damit die Gruppe beim Anlegen eines neuen Projektes als Projektleiter auswählbar ist.
Projektmitarbeiter: Aktivieren Sie diese Checkbox, damit die Gruppe beim Anlegen eines neuen Projektes als Projektmitarbeiter auswählbar ist.
Rolle für Aufgaben: Aktivieren Sie diese Checkbox, damit die Gruppe auch als Rolle in der Aufgabenverwaltung definiert ist.

Zuordnung von Benutzern zu der neuen Benutzergruppe

In der Box "Benutzer in Gruppe" sind die Benutzer aufgelistet, die der Gruppe aktuell angehören; diese Box ist zunächst leer. In der Box "alle anderen Benutzer" sind alle anderen verfügbaren Benutzer aufgelistet.
Markieren Sie einen Benutzer unter "alle anderen Benutzer" und klicken Sie auf den Button "Hinzufügen". Dieser Benutzer wird der Gruppe hinzugefügt. Der Benutzer erscheint dann unter "Benutzer in Gruppe".
Markieren Sie einen Benutzer unter "Benutzer in Gruppe" und klicken Sie auf den Button "Entfernen", damit ein Benutzer aus der Gruppe entfernt wird.
