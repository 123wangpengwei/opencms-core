The OpenCms mailing list

The opencms-dev mailing list is the place for all questions, comments and suggestions for OpenCms and the main forum of the OpenCms community. Use this list for simple "first time" user questions as well as for specialized comments, questions and/or feedback regarding OpenCms. The OpenCms core development team from Alkacon Software and several other experienced OpenCms users and developers are regularly answering questions on this list.

Participation on the opencms-dev mailing list is of course voluntary. In case you have a question where you really need an answer, there are professional support options available for OpenCms as well. Please see the Support section on the opencms.org website for details.

Archives of the OpenCms mailing list

Before you post a new question to the mailing list, you should check out the mailing list archives on the opencms.org website to see if your question has already been answered.

Using the OpenCms mailing list

To post a message to all the list members, send email to opencms-dev@opencms.org.

Important: This list requires subscription prior to posting. If you are not yet a subscriber, please use the option below to subscribe first.
