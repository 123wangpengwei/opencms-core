Hier können Sie die Sichtbarkeit / Gültigkeit von Dateien und Verzeichnissen zeitlich begrenzen. Sie definieren ein Zeitfenster, in dem die Ressource online erreichbar ist.

Geben Sie ein Anfangs- und Enddatum für das entsprechende Element ein. Sie können das jeweilige Datum manuell eingeben, wobei Sie unbedingt das Eingabeschema beachten müssen. Das oberste Feld "Datum aktualisieren" setzt das "Geändert am " Datum der Ressource, welches man im Explorer sehen kann.

Alternativ steht eine Kalenderfunktion bereit, die Ihnen eine Eingabe von Hand abnimmt. Klicken Sie hierzu am Ende der Eingabefelder auf das jeweilige Kalendersymbol und wählen mit der Maus das gewünschte Datum und Uhrzeit aus. Bestätigen Sie anschließend das Dialogfeld mit "OK". Der Datei- oder Verzeichnisname erscheint nun im Workplace in kursiver Schreibweise. Anhand dieses Merkmals sind mit Zeitfenster versehene Elemente immer gut erkennbar.

Bitte beachten Sie: Unabhängig von der Gültigkeit der Ressource können oder sollten Sie diese veröffentlichen, damit bei Eintritt der Gültigkeit in der Zukunft auch die aktuelle Version online sichtbar ist. Das Veröffentlichen beeinträchtigt die Online - Erreichbarkeit der Ressource nicht.

Tip: Wenn sich eine Ressource ausserhalb Ihrer sichtbaren Zeit im Explorer betrachten, erscheint sie kursiv.
