Durch die Verwendung dieses Menüeintrags wird eine Datei oder Verzeichnis veröffentlicht. Dies ist besonders sinnvoll, wenn  nicht das gesamte Projekt veröffentlicht werden soll, sondern nur explizit ausgewählte Elemente.

Mehr zum Thema Veröffentlichen erfahren Sie hier.

Tip: Wenn Sie mehrer Dateien in einem Verzeichnis bearbeitet haben, können Sie das übergeordnete Verzeichnis (Ordner) direkt veröffentlichen. Es werden dann alle veränderten Dateien in einem Arbeitsschritt  veröffentlicht.
