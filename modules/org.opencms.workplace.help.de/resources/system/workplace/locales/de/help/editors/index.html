Für OpenCms können verschiedene Editoren installiert werden. Ein bequemes Arbeiten ermöglichen sog. WYSIWYG-Editoren ('What you see is what you get' ), bei denen Seiten im Editor auf die gleiche Weise angezeigt werden, wie sie später dem Leser auf der Webseite erscheinen. Sie können so einfach HTML Seiten erstellen und formatieren, ohne HTML-Code verstehen zu müssen.

OpenCms unterstützt zwei WYSIWYG - Editoren:

  • MSDHTML - der Microsoft Dynamic Html für den Internet Explorer
  • HTMLArea - ein Editor für alle Browser geeigneter Editor

In manchen Fällen ist es auch von Vorteil, den HTML Quelltext direkt editieren zu können, etwa wenn die Seiten JavaScript oder speziellere HTML-Tags beinhalten, die von den Editoren nicht unterstützt werden. Für diesen Fall bietet OpenCms einen einfachen Text-Editor. XML Dokumente können einfach mit speziellen, der Dokumentenstruktur angepasste Xmlcontent - Editoren bearbeitet werden.
