Benennen Sie eine Datei oder Verzeichnis mit Hilfe dieses Menüpunkts um und / oder verschieben Sie diese im gleichen Arbeitsschritt an einen anderen Speicherort. Wenn Sie ein Element verschieben möchten, können Sie mit Hilfe der Schaltfläche „ Suchen“ das Zielverzeichnis mit Hilfe eines in einem Extrafenster dargestellten Dateibaums durch Mausklicks auswählen.

Möchten Sie lediglich ein Element umbenennen, geben sie einfach den neuen Namen in das Eingabefeld ein und bestätigen mit „ OK“. 

Hinweis: Der Name der unbenannten oder verschobenen Seite oder Datei ist nach der Aktion durchgestrichen  und bleibt bis zur Veröffentlichung erhalten. Diese „durchgestrichene Seite“ oder „ durchgestrichene Datei“ kann jederzeit wieder hergestellt werden. Erst nach dem veröffentlichen  wird die durchgestrichene datei entgültig gelöscht. 

Beachten Sie bitte: Leer- und Sonderzeichen dürfen nicht verwendet werden.
