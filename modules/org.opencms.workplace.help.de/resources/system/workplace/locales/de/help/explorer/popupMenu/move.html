Benennen Sie eine Datei oder Verzeichnis mit Hilfe dieses Menüpunkts um und / oder verschieben Sie diese im gleichen Arbeitsschritt an einen anderen Speicherort. Wenn Sie ein Element verschieben möchten, können Sie mit Hilfe der Schaltfläche „ Suchen“ das Zielverzeichnis mit Hilfe eines in einem Extrafenster dargestellten Dateibaums durch Mausklicks auswählen. Wenn Sie so ein Verzeichnis wählen, so wird der Name der Datei nicht verändert, sondern die gewählte Datei in das Zielverzeichnis verschoben.

Möchten Sie lediglich ein Element umbenennen, geben sie einfach den neuen Namen in das Eingabefeld ein und bestätigen mit „ OK“. Dann wird im aktuellen Verzeichnis die Datei umbenannt.

Hinweis: Der Name der unbenannten oder verschobenen Seite oder Datei ist nach der Aktion durchgestrichen  und bleibt bis zur Veröffentlichung erhalten. Diese „durchgestrichene Seite“ oder „ durchgestrichene Datei“ kann jederzeit wieder hergestellt werden. Erst nach dem veröffentlichen  wird die durchgestrichene datei entgültig gelöscht. 

Hinweis: Leer- und Sonderzeichen dürfen nicht verwendet werden.

Tip: Zum gleichzeitigen Verschieben und umbenennen können sie mit der o.g. Suchfunktion ein Zielverzeichnis auswählen und anschliessend in der Textbox noch einen neuen Dateinamen eingeben.
