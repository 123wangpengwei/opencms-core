Auf dieser Karte können Sie Ihre Benutzerdaten überprüfen und ggf. abändern. Alle Änderungen die Sie machen werden von OpenCms gespeichert und sind immer noch gültig, wenn Sie sich das nächste mal anmeldnen. Viele Einstellungen erlauben es Ihnen, komfortabler und schneller zu arbeiten. Wenn sie z. B. länger nur an einem Projekt arbeiten und dort auch nur in einem bestimmten Verzeichnis einer bestimmten Site, so können Sie OpenCms so einstellen, dass sie nach der Anmeldung sofort mit der Arbeit beginnen können: In der richtigen Site, dem richtigen Projekt am richtigen Pfad im Explorer. Ebenfalls können Sie ihre bevorzugte Sprache auswählen. Weiterhin können Sie auf dieser Registerkarte ihr Kennwort ändern.

Über diese Schaltfläche erreichen sie Ihre benutzerspezifischen Einstellungen.

Das Layout ist durch "Karteikarten mit Reitern" oder auch "Tabbed Pane" realisiert. Sie klicken auf einen Tab und die zugehörige Seite mit den Einstellungen wird sichtbar.

"Workplace"-Tab

  • Sprache:
    Hier können Sie Ihre Sprache einstellen. OpenCms 6.0.0 wurde so entwickelt, dass sich nachträglich neue Sprachpakete (auch von Programmierlaien) ohne eine steile Lernkurve auch von Dritten entwickeln lassen können. Diese können dann nachträglich als Module in OpenCms installiert werden. In der Grundversion "spricht" OpenCms und bringt das deutsche Sprachmodul bei der Installation mit. Somit lassen sich hier vorerst nur "en" und "de" (die in ISO 639 definierten Sprachcodes) auswählbar.
  • Starte mit Projekt:  
    Hier können Sie per "ComboBox" aus einer ausziehbaren Liste eines der in OpenCms erstellten Projekte auswählen. Wenn Sie sich das nächste mal in OpenCms anmelden werden sie in diesem Projekt beginnen.
  • Starte mit Ansicht :
    Hier können Sie auswählen, in welcher Ansicht Sie nach dem Anmelden an OpenCms beginnen wollen, meist werden sie den Explorer auswählen:
  • Starte in Site :
    Hier wählen Sie die Site, in der Sie sich nach dem Anmelden befinden werden. Mehr über das Konzept von Sites finden sie hier.
  • Starte in Ordner :
    Hier wählen Sie den Ordner, in dem Sie sich befinden werden, wenn sie zur Explorer Ansicht wechseln oder sich anmelden und die Ansicht "Explorer" als Starteinstellung gewählt haben. 
  • Workplace Buttons :
    Sie können wählen ob die oberste Titelleiste der Workplace, die in jeder Ansicht vorhanden ist, nur Bilder, nur Text oder beides enthalten soll. Eine Geschmackssache, die auch Puristen, die häufig redaktionell mit OpenCms arbeiten (also die Bezeichnungen kennen), zufriedenstellt.
  • Report Anzeige :
    Hier kann der Detailgrad der Reports minimiert werden. Immer, wenn sie ein Modul importieren oder exportieren, Links überprüfen lassen, OpenCms installieren, ... sehen Sie Html Reports mit - per Standard - ausführlichen Informationen. Applet zum Hochladen Verwenden :
    Wählen Sie aus, in welcher Form Sie Dateien auf den Server laden wollen. Mehr Details über die Unterschiede erfahren Sie hier.
