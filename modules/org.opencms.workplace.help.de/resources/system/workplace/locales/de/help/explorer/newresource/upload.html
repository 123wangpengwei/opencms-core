Mit Hilfe dieses Menüpunktes können Sie Dokumente, Bilder und andere Dateien auf den OpenCms Server laden und so Ihren Lesern beispielsweise als Download bereitstellen. Es gibt zwei Möglichkeiten des Hochladens, die beide Ihre Vorteile haben:

  • Hochladen durch Ihren Browser
  • Hochladen mit einem Java Applet

Welche Methode Sie zum Hochladen verwenden, konfigurieren Sie in Ihren Benutzereinstellungen.  

Diese Aktion kann entweder über die Schaltfläche der Titelleiste oder über die Schaltfläche "Neu" mit der Option "Neue Datei hochladen" gestartet werden.
