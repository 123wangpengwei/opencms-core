Um eine bestehende Verknüpfung zu editieren, ist lediglich ein Klick auf das entsprechende Symbol notwendig. Im Kontextmenü finden Sie den Menüpunkt „Verknüpfung editieren“. Starten Sie den Dialog mit einem weiteren Klick und ändern die URL entsprechend den Erfordernissen. Bestätigen Sie anschließend mit OK.

Kontextmenüeintrag zum Editieren einer Verknüpfung: OpenCms nennt den Dateityp "pointer".
