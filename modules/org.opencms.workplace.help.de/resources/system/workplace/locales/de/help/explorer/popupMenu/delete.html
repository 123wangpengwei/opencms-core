Löscht eine Datei oder ein Verzeichnis aus dem Projekt. Nach dem Löschvorgang erscheint das Element durchgestrichen im Workplace und wird erst nach einem Veröffentlichen vollständig aus Ihrem Projekt entfernt.

Ein gelöschtes (und damit durchgestrichen dargestelltes) Element können Sie wiederherstellen, in dem Sie nach einem Klick auf das Datei- oder Verzeichnissymbol den Menüeintrag „ Wiederherstellen“ auswählen.

Bitte beachten Sie: Sie können den Löschvorgang nur wieder Rückgängig machen, wenn Sie seit dem Löschen die Datei (oder das zug. Projekt) nicht wieder veröffentlichen.
