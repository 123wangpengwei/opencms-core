Aktiviert eine verschlüsselte Übertragung für die ausgewählte Ressource. Eine verschlüsselte Übertragung von Inhalten ist immer dann sinnvoll, wenn z.B. persönliche Informationen eines Users an den Webserver übermittelt werden. Insbesondere sind hier E-Mail-Formulare zu nennen.

Nach einem Klick auf den Menüeintrag „Sichern / Exportieren“ öffnet sich ein Dialogfenster in dem die entsprechenden Einstellungen vorgenommen werden können. 

  • Verschlüsselt übertragen via HTTPS
     Wird hier der Wert „Ja“ gesetzt, wird die Datei bei einem Aufruf durch einen Leser verschlüsselt übermittelt, sofern Sie über die Seiten- oder Kopfnavigation aufgerufen wird. Erkennbar ist eine verschlüsselte Seite an dem https:// in der Adresszeile des Browsers. Auch in der Statuszeile des Browsers wird auf die SSL-Verschlüsselung hingewiesen.
  • Intern       
     Wird in dieses Auswahlfeld das Häkchen gesetzt, ist die entsprechende Datei nur noch intern, sprich durch einen OpenCms-Benutzer aufrufbar. Sie können die Seite oder das Dokument nur über den Workplace aufrufen, nicht aber direkt über das Internet. Damit können z.B. die Seitenelemente des Templates vor einem direkten Aufruf geschützt werden:  Wenn diese Funktion am Seitenelement aktiviert wurde, dann kann der Inhalt des Seitenelements nur noch gemeinsam mit dem Template aufgerufen werden.
