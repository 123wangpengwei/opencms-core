Die Funktion „Historie“ erlaubt es, ältere veröffentlichte Versionen einer Datei wiederherzustellen. Das Dialogfenster zeigt die Historie der veröffentlichten Dateiversion. Sollte durch Bearbeiten, Speichern und anschließendes Veröffentlichen einer Datei ein noch benötigter Inhalt verloren gegangen sein, haben Sie mit Hilfe der Historie einen Zugriff auf die älteren Versionen dieser Datei. Nach jedem Veröffentlichen wird die zuvor aktuelle Version vom System nicht verworfen, sondern in der Historie archiviert.

Ein Klick auf den Menüeintrag „Historie“ ruft den entsprechenden Dialog auf und zeigt Ihnen die archivierten Versionen an. Anhand der Versionsbeschreibung (Veröffentlichungsdatum, letzte Bearbeitung...) können Sie die gewünschte Version markieren. Es werden nicht alle, sondern nur eine bestimmte, frei konfigurierbare Anzahl von Versionen (standartmäßig 10) gespeichert.


Wiederherstellen einer älteren Datei-Version mit Hilfe der Historie

 

Ein Klick auf dieses Symbol öffnet ein Fenster mit dem Inhalt dieser Version. Leider ist eine normale Layout-Ansicht derzeit nicht möglich. Es handelt sich hierbei um eine Art Quelltextansicht zur Kontrolle der Inhalte.
Haben Sie mittels der Vorschaufunktion die gewünschte Version ausfindig gemacht, können Sie diese mit Hilfe dieser Schaltfläche wiederherstellen.
