Sie können auch ohne zusätzliche Software Dateien auf den Server des Erzbistums hochladen.

Dies ist besonders dann sinnvoll, wenn Sie:

  • nur gelegentlich ein paar wenige Dateien auf den Server hochladen möchten
  • über eine schmalbandige Internetverbindung (Modem, ISDN) verfügen

Wollen Sie mehrere Dateien auf einmal auf den Server hochladen, müssen diese zuvor als Zip-Datei zusammengefasst werden. Im Dialogfenster „Datei hochladen“ müssen Sie dann die zusätzliche Option “Zip-File entpacken“ das Häkchen setzen.

Nachteile:

  •  Langsamerer Datentransport
  • Nur eine Datei pro upload (oder mehrere Dateien vorher zu einem zip - archiv zusammenfassen)

Vorteile:

  • Keine Notwendigkeit weitere Software zu installieren.
  • Ihr Browser merkt sich, von welchem Ort auf Ihrer Festplatte aus Sie das letzte mal hochgeladen haben. Wenn Sie also häufig einzelne Dateien hochladen, sparen Sie sich Zeit durch das Suchen Ihrer Dateien zum Hochladen.

Diese Variante bietet weit weniger Komfort als die zuvor beschriebene, ist aber für das „gelegentliche“ hochladen von Dateien auf den Webserver deutlich schneller und völlig ausreichend.

Tip:Bei Windows XP können Sie Dateien ohne Zusatzsoftware als Zip-Archiv zusammenfassen. Markieren Sie die gewünschten Dateien und klicken anschließend mit der rechten Maustaste auf die Auswahl. Im Kontextmenü wählen Sie nun 4Senden an 4Zip-komprimierter Ordner.

Bei älteren Windows-Versionen (98, ME oder 2000) ist eine Zusatzsoftware nötig. Diese können Sie kostenlos unter www.winzip.de herunterladen.
