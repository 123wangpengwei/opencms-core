Dieser Dialog bietet keine immanent wichtigen Einstellungen sondern etwas Komfort. Sie können hier Standardeinstellungen (Vorbelegungen) für das Erstellen neuer Projekte festlegen. Wenn Sie dann später ein neues Projekt anlegen, werden die hier eingestellten Werte in den Dialog zum Erstellen eines neuen Projektes eingetragen.

Die Wahl der "Anzeige von Projektressourcen" wirkt sich anders als die übrigen Einstellungen nicht auf den Dialog zum Erstellen eines neuen Projektes aus, sondern manipuliert, welche modifizierten Ressourcen (geänderte Dateien, neue Dateien, gelöschte Dateien, alle Änderungen) sie sehen wollen, wenn sie im Hauptdialog für das Projektmanagement auf einen Projektnamen oder einen Eintrag in der Spalte ganz links klicken.
