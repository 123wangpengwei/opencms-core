Mit Diesem Dialog Sie können sie Standardeinstellungen (Vorbelegungen) für das Erstellen neuer Projekte festlegen. Wenn Sie dann später ein neues Projekt anlegen, werden die hier vorkonfigurierten Werte in den Dialog zum Erstellen eines neuen Projektes eingetragen.

Der Dialog für die Projekteinstellungen

 

Die Wahl der "Anzeige von Projektressourcen " wirkt sich anders als die übrigen Einstellungen nicht auf den Dialog zum Erstellen eines neuen Projektes aus, legt fest, welche modifizierten Ressourcen (geänderte Dateien, neue Dateien, gelöschte Dateien, alle Änderungen) angezeigt werden, wenn sie im Hauptdialog für das Projektmanagement auf einen Projektnamen oder einen Eintrag in der Spalte ganz links klicken.
