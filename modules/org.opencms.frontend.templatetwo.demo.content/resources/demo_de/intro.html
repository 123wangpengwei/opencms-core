Template Two ist ein konfigurierbares Template, das es ermöglicht, auch eigene Websites unter Verwendung dieses Templates relativ einfach zu erstellen und anhand der Konfigurationeinstellungen (hier im Verzeichnis _config_) den jeweiligen Erfordernissen anzupassen.

Beachten Sie hierbei auch, daß in einigen Spezialfällen auch so genannte "Properties" herangezogen werden, insbesondere um Standardeinstellungen zu definieren. Die Properties können über das Kontextmenü jeder Ressource bearbeitet werden; ist kein Wert direkt gesetzt, so wird er u.U. entlang der Verezeichnishierarchie eingelesen.

<![CDATA[Introduction]]> ac046b97-e156-11dc-a180-9f7a6a6ce285 9b18e429-e448-11dc-add5-9f7a6a6ce285 These demo pages are used to  show  various features of OpenCms using pages based on the OpenCms template  "Template Two" as an example. 

 

Template Two is a configurable template intended to be used as a starting point for developing your own web site with OpenCms. By modifying the configuration settings (in the _config_/layout directory), you can adapt it to your own requirements.

 

At each page in the left column, you will find a red box used for explaining the elements of the page. In order to understand you should not only look at the pages, but open them with th eeditor in the OpenCms workplace.

 

Please keep in mind, that in some special cases, there are additional settings available using OpenCms "properties", especially for setting default values. These properties can be edited via the contextual menu of each resource - if there is not value set at the reosurce, immediately, the value is serached along the folder hierarchie.
