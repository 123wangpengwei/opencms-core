Dynamic include of additional javascript and css:
 

1) Edit template file:

/system/modules/org.opencms.dev.demo/templates/main.jsp

Add new tag to the template <head>:



<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN“ “…">
<html><head>
    […]

    <cms:headincludes type="javascript" />
    <cms:headincludes type="css" />

     […]
</head><body>
[…]

 

2) Edit the xml schema definition:

/system/modules/org.opencms.dev.demo/schemas/article_with_headincludes.xsd

Add new nodes:

[..]
<xsd:annotation>
<xsd:appinfo>
[…]
<headincludes>

     <headinclude type="javascript” uri="..." />
     <headinclude type="css” uri="...“/>

</headincludes>
</xsd:appinfo>
</xsd:annotation>
 

3) Use ${cms.edited} variable:

Define a specific areas inside of the formatter to remind the user to reload the page after editing.

<c:if test="${cms.edited}">
        ...
</c:if>
