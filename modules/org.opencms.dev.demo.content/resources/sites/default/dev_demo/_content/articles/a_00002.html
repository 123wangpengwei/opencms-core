<![CDATA[Dev Demo Module]]> The Development Demo module (Dev Demo) is a collection of use cases for OpenCms 8. 

The demo examples have similar structure:

  • Each demo example has its own sub sitemap
  • The path to the sub sitemap of a demo example: /sites/default/dev_demo/{example_name}/_config/
  • The path to the content of a demo example: /sites/default/dev_demo/{example_name}/_contents/
