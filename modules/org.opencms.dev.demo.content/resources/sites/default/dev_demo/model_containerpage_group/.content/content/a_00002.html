<![CDATA[How to use the group container]]>
  • Open the group container editor by cliking on the edit button of the group element. The group container editor appears in an overlay. Now only the group container elements and the title and description property can be edited.
  • Drag and drop, edit or delete the existing elements inside of the group container.
  • Drop a new element to the group container.
  • Save the changes by clicking ok on the group container editor.

    •  

    The changes to the group container elements are populated to all pages using the page model.
