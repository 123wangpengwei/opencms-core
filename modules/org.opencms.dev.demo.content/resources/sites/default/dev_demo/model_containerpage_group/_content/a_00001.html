<![CDATA[Group Container Demo]]> A Group Container is a content element referencing a group of other content elements. A Group Container is rendered as if it’s sub elements where referenced directly by the container page.
It allows to maintain the referenced elements in one place to take effect on many pages.
 

Add a group container to the model container page, if you would like to have following effects on your website:

  • A common content part, which is repeated on all pages
  • All changes (edit, new and delete action) inside of the group container will effect all pages.
